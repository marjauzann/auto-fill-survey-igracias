### Langkah-langkah:

1. **Buka halaman survey iGracias**
   - Login ke iGracias Telkom University
   - Buka halaman survey yang ingin diisi

2. **Buka Developer Console di browser**
   - **Chrome/Edge**: Tekan `F12` atau `Ctrl + Shift + J` (Windows/Linux) / `Cmd + Option + J` (Mac)
   - **Firefox**: Tekan `F12` atau `Ctrl + Shift + K` (Windows/Linux) / `Cmd + Option + K` (Mac)
   - **Safari**: Tekan `Cmd + Option + C` (perlu enable Developer menu dulu)

3. **Copy script**
   - Buka file `main.js`
   - **Select All** (Ctrl+A / Cmd+A) dan **Copy** (Ctrl+C / Cmd+C)

4. **Paste ke Console**
   - Klik pada area console (tempat mengetik)
   - **Paste** script (Ctrl+V / Cmd+V)
   - Tekan **Enter**

5. **Selesai!**
   - Form akan otomatis terisi
   - Anda bisa langsung klik tombol "Simpan" atau "Submit"

## ğŸ”§ Troubleshooting

### Script tidak bekerja?
1. Pastikan console tidak ada error (warna merah)
2. Pastikan halaman sudah sepenuhnya dimuat
3. Coba refresh halaman dan jalankan script lagi

1. Cek apakah ada radio button atau text input di halaman
2. Pastikan form belum di-submit
3. Coba jalankan script sekali lagi

- Pastikan Developer Tools sudah dibuka (F12)
- Cek apakah tab "Console" sudah dipilih
- Beberapa browser mungkin perlu diaktifkan Developer Tools terlebih dahulu
